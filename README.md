## How to run 🏃
1. Clone the repo

        git clone https://github.com/reenphygeorge/MyDiary

2. Install the required dependencies
          
        npm install
        
3. Now start the server

        nodemon app.js
        
4. Now open any browser and search
  
         http://localhost:5000/home
         
5. That's All :relieved:
